# Prefecture fortune
## 概要
株式会社ゆめみ iOS エンジニアコードチェック課題
テーマ「あなたと相性のいい都道府県を占ってあげる！」

## アプリの仕様
ユーザから名前，生年月日，血液型の情報を受け取り，以下のAPIに問い合わせ
"https://yumemi-ios-junior-engineer-codecheck.app.swift.cloud"
返ってきた結果を表示する。
